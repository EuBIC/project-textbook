# EuBIC Computational MS Textbook

> A community-driven, open, living textbook on **computational mass spectrometry**, initiated by [EuBIC-MS](https://eubic-ms.org).

---

## 📖 About

Current educational resources on computational mass spectrometry are either outdated, scattered, or not written with students and early-career researchers in mind. This project aims to change that.

Our goal is to build a **free, open, and continuously updated** textbook that:

- Supports existing and new university courses on computational MS
- Serves as a reference across tools, workflows, and research
- Grows with the field - not a one-time snapshot, but a living document
- Is accessible to both biologists and informaticians

This is a **community effort**. Contributions, reviews, and discussions from anyone in the field are welcome.

If you like to help, please [join EuBIC-MS](https://eubic-ms.org/become-a-member/) and the [#project-textbook](https://eubic.slack.com/archives/C02HQ1EJA57) channel.


## 🛠️ Technical Approach

We are evaluating the following tools for collaborative writing and publishing:

| Tool | Notes |
|------|-------|
| [Manubot](https://github.com/manubot/manubot) | Used by the [Proteomics Tutorial](https://jessegmeyerlab.github.io/proteomics-tutorial/); outputs HTML and PDF |
| [Quarto](https://quarto.org/) | Supports interactivity; mix of Markdown and Jupyter notebooks |
| [JupyterBook](https://jupyterbook.org/) | Similar to Quarto; notebook-native |

Feedback on tooling is welcome — open or join Discussion!
